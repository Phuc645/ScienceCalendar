# Dự án ScienceCalendar

## Tổng quan

Đây là một website xem lịch, mỗi ngày hiện ra một fact về ngành khoa học tự nhiên dưới lịch. Các fact sẽ đi theo: Toán -> Lý -> Hoá -> Sinh. Background web cũng sẽ thay đổi theo fact, với Toán là màu xám, Lý là màu xanh dương, Hoá là màu hồng, còn Sinh là màu xanh lá cây.

## Thời gian dự kiến:

Khoảng từ 3 đến 4 tháng

## Công nghệ

- **Frontend**: React
- **Backend**: ExpressJS
- **Database**: PostgreSQL
- **Deployment**: Vercel, Render
